⚙️ Configurar Supabase

Para que a plataforma funcione em tempo real com múltiplos utilizadores, precisas de configurar o Supabase (gratuito). Segue estes passos:

1 Criar conta: Vai a supabase.com e cria uma conta gratuita.
2 Criar projeto: Cria um novo projeto (dá-lhe qualquer nome, ex: "merchandise-hyundai").
3 Criar tabelas: Vai a SQL Editor no menu lateral e cola o SQL que está no ficheiro SETUP_SQL.txt que te foi fornecido. Clica em Run.
4 Copiar credenciais: Vai a Settings → API e copia o Project URL e a anon public key.

As credenciais ficam guardadas no teu browser. A anon key é pública e segura para uso no frontend.
